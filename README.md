# Repositório de Exercícios em Java

Este repositório foi criado para organizar e documentar minha jornada de aprendizado em **Java**.  
Aqui você encontrará exercícios práticos, exemplos de código e pequenos projetos que refletem minha evolução no curso.

## 🎯 Objetivos
- Consolidar conceitos fundamentais de Java (variáveis, loops, arrays, etc.)
- Praticar **Programação Orientada a Objetos** (POO) com exemplos de herança, polimorfismo e encapsulamento
- Desenvolver mini-projetos aplicados à realidade angolana, como sistemas de gestão e cálculos em Kwanza (Kz)
- Criar uma base de código organizada e reutilizável para futuros projetos maiores

## 📂 Estrutura
- **fundamentos/** → exercícios básicos
- **poo/** → conceitos de orientação a objetos
- **colecoes/** → uso de List, Set e Map
- **projetos/** → mini-aplicações práticas
- **testes/** → exercícios de revisão e testes automatizados

## 🚀 Motivação
Este repositório é parte da minha formação em Java e também serve como portfólio para mostrar minha dedicação e capacidade de aplicar tecnologia em contextos reais de Angola.
